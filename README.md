E-Library Management System (Django)

Developed a web-based E-Library Management System using Django to automate book management and user operations.

Implemented secure user authentication and role-based access for Admin and Users.

Enabled core features such as book search, issue, return, and availability tracking.

Designed relational database models to manage books, users, and transactions efficiently.

Built a responsive user interface using HTML, CSS, and Bootstrap.

Tech Stack: Python, Django, HTML, CSS, Bootstrap, SQLite/PostgreSQL

How to run this Library Management System Project

1. Download and Unzip file on your local system copy library.
2. Put library folder inside root directory

Database Configuration

Open phpmyadmin
Create Database library
Import database library.sql (available inside zip package)
